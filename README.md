# ecommerce_api
Ecommerce Case - API usgin Node.JS

Main Modules:
* Sequelize
* Mysql
* Express
* Jsonfile
* Nodemon
* Supertest / Joi / Chai
* Babel

Install Application
    npm install

Run Application: 
    npm start
    npm run start-dev (nodemon)

Run Tests: 
    npm test (integration, unit and contract)
